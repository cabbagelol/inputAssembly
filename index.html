    返回结果

    在控件中获取表单结果使用check_result()方法取得，方法中提供了一个参数，仅仅向内部传递'方法'(下方表格),当然inpitassembly是可以指定获取表单，如果你需要这样做，则在check_result()方法传入表单名称或是表单下任意对象，参考下面"返回规则"说明.

    [更新]按钮

    [2.8]使用check_result方法实在太麻烦,你可以创建一个'type=result'按钮(必须是button)在这个表单下(任意位置,可在选项内),它同等于check_result($(this)),你必须记住返回的结果是该按钮所在表单结果，如果内部重叠其他表单，那么返回结果是它含自己以及所有子表单结果；而这些结果都回通过success回调

    HTML
    
<button type="result">获取结果</button>
					
    HTML
    
<button type="reset">重置</button>

    [更新]多表结果

    [2.8]你可以通过check_result方法取得同一个页面上不同表单的结果,通过逗号(’,‘)划分需要获取表单，为了让你清楚check_result获取结果规律我将排列成一个表格:

    单个表单 嵌套表单 兄弟表单
    check_result( $(this) ) √
    check_result( 'name' ) √ √ √
    check_result( 'name,name' ) √ √ √
    • 单个表单: 可以通过check_result( 'name' )方法来指定，像check_result( $(this) )通常是取当前点击按钮所属表单结果，因此它仅能取得单个表单结果
    • 嵌套表单: 表单内嵌套多层表单，如果获取父表单，子表单也被包含在结果中，例如check_result( 'name' )将索引出表单并取出它的结果，这代表只要有符合表单名便遍历结果
    • 兄弟表单: 通常在同一标签内，check_result( 'name' )和check_result( 'name,name' )方法都可以取得，它们可以跨过嵌套之间的限制，因此除check_result( $(this) )方法外都是全局索引表单取出结果

    变动注意

    • 现在'check_result'似乎有点不同，新版本中返回结果也会在success中返回，你依旧可以在任意地方使用check_result返回的结果直接处理；也可以像这样在success中注册获取结果按钮，再通过success回调内容处理。 

      JS
      
$().inpitassembly({
	success:function(res){
		// * =====================
		// * 注册按钮
		// * =====================
		
		// 为避免重复注册可以通过判断success内回调内容是否定义来决定
		
		$("button").on("tag click",()=>{
			check_result("radioa")
		})
		
		// * =====================
		// * 当触发绑定的button事件处理success回调内容
		// * 处理res变量结果
		// * =====================
		for(i of ...res.data.data){
			//...
		}
	}
});

$("[type='inpit/assembly']").find("button").click(function(){

	// * ==========================

    你会发现使用单选会触发二个消息，它们分别是移除和添加，这是因为首先会移除上一个选项在添加当前选项

    JS
    
通过on方法返回事件，在inpitassembly方法内on会回调事件以方便监听表单下发生什么.
在inpitassembly中，包含2个参数，第一个参数返回状态(add/rem/disabled);第二个参数返回对象，如下：

$().inpitassembly({
	on:function(e,o){
		$(".cp_fiveon").prepend("<div>" + (function(){
			let val = e;
			if(val == "add")
				val = "选中" + o.this.obj.attr("value");
			else if(val == "rem")
				val = "移除";
			else if(val == "disabled")
				val = "选项禁用";
				
			return val;
		})() + "，所属" + (function(){
			let type = o.this.type;
			if(type == "radio"){
				type = "(复选)"
			}else if(type == "checkbox"){
				type = "(单选)"
			}
			return type;
		})() + "表单：" + o.this.parent.attr("formname") +  "</div>"); 
	}
});

    JS

// * 兼容:2.8

$().inpitassembly({
	event_:{
		state:true, 
		// 默认下是false，启动它必须设置为true，不能忽略
		class:"*",
		// 这里是通配符，可以写成".class"或".class,.classb"、".class > div"
		// 如果class缺省，state则自动变为false
		complete: function (event) {

		}
	}
})
